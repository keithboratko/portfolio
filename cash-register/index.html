Welcome to Cash Register!

Fill out the input fields, where "Price" is the price of an item, "Cash" is the amount used to pay for the item, and each specific denomination amount tallies the total amount of each specific denomination (e.g. 100 pennies in register = "1.00", 2 quarters in register = "0.5"). If the total amount for a denomination is less than 1, be sure to express the value as "0.__".

When submitted, a function will run to calculate appropriate change (in the most efficient way) if possible, and returns an array displaying the change made, as well as whether or not the register is "Open" or "Closed" after this purchase.

If making change is not possible because either a) the register total is less than the change needed, or b) making change for the amount needed is not possible with the denominational breakdown of the register, the function will return a status of "INSUFFICIENT FUNDS", followed by an empty array.
